# Practical-react-exercises

### **Task 1. Counter App**

**Task:** Make an app that counts numbers.

**What to do:**

- Start with the number 0.
- Add two buttons:
    - One button increases the number by 1.
    - Another button decreases the number by 1, but the number cannot go below 0.
- Show the number on the page.
- Style the buttons to look nice and easy to click.

### **2. Real-time Input**

**Task:** Make a form that shows what the user types right away.

**What to do:**

- Add a text box for the user to type in.
- As the user types, show the text below the box immediately.
- Use React’s useState to handle the typing.
- Make the text easy to see with some simple styling.


### **Task 3:** Create a list that shows items.

**What to do:**

- Make a React component that takes a list (array) of items.
- Show the items as a numbered list or a regular bullet list.
- Add a hover effect (e.g., change the color) when the user points at an item.
- Add a button to let the user switch between numbered and bullet lists.


### **4. Toggle Switch**

**Task:** Create a button that switches between "On" and "Off".

**What to do:**

- Make a toggle switch with two states: "On" and "Off".
- Use useState to handle the switching.
- Show different colors for each state (e.g., green for "On" and gray for "Off").
- Style it to look like a real switch, with smooth animation when clicked.

### **5. API Data Fetch**

**Task:** Fetch and show data from the internet.

**What to do:**

- Use React’s useEffect to get data from a public API (like JSONPlaceholder).
- Show the data as a list, table, or cards.
- Show a loading message or spinner while fetching the data.
- If something goes wrong, show an error message.
- Add some basic styling to make the data easy to read.

### **6. Calculator App**

**Task:** Make a simple calculator.

**What to do:**

- Add a way for users to type numbers.
- Include buttons for basic math operations (+, -, *, /).
- Show the result right away when the user clicks a button.
- Add a "Clear" button to reset everything.
- Style it to look like a calculator, with clear buttons and numbers.

### **7. Timer**

**Task:** Build a countdown timer.

**What to do:**

- Let users type how many seconds the timer should count down.
- Add a "Start Timer" button to begin.
- Show the time left on the page and update it every second.
- When the time reaches 0, show "Time’s up!".
- Add "Pause" and "Reset" buttons.
- Style the timer and buttons to look clean and easy to use.

### **8. To-Do List**

**Task:** Make an app to manage tasks.

**What to do:**

- Add a text box and a button labeled "Add Task".
- When the user adds a task, show it in a list below.
- Each task should have:
    - A checkbox to mark it as done (cross it out when done).
    - A "Remove" button to delete it from the list.
- Make the list look nice with styling, so it’s easy to read and use.
